This is a paragraph. You can add emphasis to text by using _asterisks_ or _underscores_. For example, _this text will be italicized_ and _this text will be italicized_.

You can also make text **bold** by using **double asterisks** or **double underscore**. For example, **this text will be bold** and **this text will be bold**.

You can create lists by using hyphen (-) or asterisks (\*). For example:

- Item 1
- Item 2
- Item 3

Or:

- Item 1
- Item 2
- Item 3

You can also create ordered lists by using numbers. For example:

1. First item
2. Second item
3. Third item

You can add links by using bracket [] and parentheses (). For example:
[Google](https://www.google/com) will create a link to Google.

You can add images by using an exclamation mark (!), brackets [], and parentheses (). For example:

## Lists

Unordered
