## Diodos

Formulas

$$
\begin{gathered}
	\text{Rectificador de Media Onda}\\
	I\_D = \frac{V\_I-V\_\gamma}{R} \\
	V\_O = (I\_D)(R) = V\_I-V\_\gamma \\\\
	V\_{O(avg)} = f \int^{T}\_{0} V_O(t) dt = \frac{1}{T} \int^{T}\_{0} V_O(t) dt\\
\end{gathered}
$$

**2.1** Considere el circuito de la imagen, sabemos que $R=1k\Omega$, $V_\gamma = 0.6V$, $r_f = 20\Omega$. Graficar $V_O - V_I$ de $V_I=-10V$ hasta $V_I=10V$. Después asuma que $V_I=10\sin{(wt)}V$. Grafique $V_O - t$. Calcule $V_{O(avg)}$, $I_{D(max)}$ y el PIV.   

![Circuito](/static/images/electronicaproblemas/2.1.png)

$$
\begin{gathered}
	V\_I < 0.6V \implies V\_O = 0\\
	V\_I > 0.6V \implies V\_O = (I\_D)(R) = (\frac{V\_I-V\_\gamma}{R + r\_f})(R)\\
	V\_O = \frac{V\_I-0.6}{1000+20}(1000)\\
	V\_O(10)=\frac{10-0.6}{1000+20}(1000) = 9.2157V
\end{gathered}
$$

![Circuito](/static/images/electronicaproblemas/2.11.png)

$$
\begin{gathered}
	V\_O = (I\_D)(R) = (\frac{V\_I-V\_\gamma}{R + r\_f})(R)\\
	V\_O = (\frac{(10\sin{(wt)})-0.6}{1000 + 20})(1000) = 0\\
	10\sin{(wt)}-0.6 = 0 \implies \sin{(wt\_1)} = \frac{0.6}{10} = 0.06\\
	x\_1 = (wt\_1) = \sin^{-1}{(0.06)} = 3.439° =  0.060036 rad\\
	x\_2 = (wt\_2) = 180° - 3.439° = 176.560° = 3.081556 rad\\
	V_{O(avg)} = \frac{1}{T} \int^{T}\_{0} V_O(t) dt = \frac{1}{2\pi} \int^{2\pi}\_{0} (10\sin{(x)}-0.6) dx\\
	V_{O(avg)} = \frac{1}{2\pi}(( (-10\cos{176.560°}) - (-10\cos{3.439°}) ) + ((-0.6(3.081556))-(-0.6(0.060036))))\\
	V_{O(avg)} = 2.888V\\
\end{gathered}
$$

**2.2** Demostrar que si el voltaje de entrada es mayor o igual a 0 para el circuito anterior entonces el voltaje de salida es aproximadamente $V_O=V_I-V_Tln(\frac{V_O}{I_SR})$.

$$
\begin{gathered}
	\text{LKV}\\
	V\_O = V\_R = V\_I - V\_D\\ 
	\text{Diodo Ideal}\\
	V\_D = (V\_T)(ln\left(\frac{i\_D}{I\_S}\right)\\
	\therefore V_O \approx V\_I-V\_T ln(\frac{V\_O}{I\_SR})
\end{gathered}
$$

En este problema recordamos la ecuación para el voltaje del diodo ideal ignorando el menos uno a que solo estamos considerando el voltaje positivo.

**2.3** Considere un rectificador de media onda como en la imagen, asuma que $R=2k\Omega$, $N = 10:1$, $V_\gamma=0.7$ y $V_I= 120V(rms) \land 60Hz$. Calcular voltaje de salida máximo, corriente máxima del diodo, porcentaje del ciclo en donde el voltaje de salida es mayor a 0, el voltaje de salida promedio y la corriente promedio del diodo.

$$
\begin{gathered}
	V\_I = 120V(rms) = 120(\sqrt(2)) \approx 169.7056V(p)
\end{gathered}
$$

![Circuito](/static/images/electronicaproblemas/2.3.png)

$$
\begin{gathered}
	V\_{S(max)} = 169.7056(\frac{1}{10}) = 16.9705V\\
	V\_{O(max)} = V\_I - V\_D = 16.9705 - 0.7 = 16.2705V\\
	I\_{D(max)} = \frac{V\_{O(max)}}{R} = \frac{16.2705}{2000} = 0.008135 A\\
	V\_{O} = (V\_{S(max)} \sin{(wt)}) - 1 (V\_\gamma) = (16.9705\sin{(wt)})-(0.7)\\
	V\_{O} = 0 \implies (16.9705\sin{(wt)})-(0.7) = 0 \implies \sin{(wt)} = \frac{0.7}{16.9705} = 0.04125\\
	x\_1 = (wt\_1) = \sin^{-1}{(0.04125)} = 2.364° = 0.041262 rad\\
	x\_2 = (wt\_2) = 180° - 2.364° = 177.636° = 3.100331 rad \\
	\text{Porcentaje } = \frac{177.636-2.364}{360} \times 100\\% = 48.7\\%\\
	V\_{O(avg)} = \frac{1}{T} \int^{T}\_{0} V_O(t) dt = \frac{1}{2\pi} \int^{x\_2}\_{x\_1} (16.9705\sin{(x)}-0.7) dx \\
	V_{O(avg)} = \frac{1}{2\pi}(( (-16.9705\cos{177.636°}) - (-16.9705\cos{2.364°}) ) + ((-0.7(3.100331))-(-0.7(0.041262))))\\
	V\_{O(avg)} = 5.056V\\
	I\_{D(avg)} = \frac{V\_{O(avg)}}{R} = \frac{5.056}{2000} = 0.00253 A
\end{gathered}
$$
